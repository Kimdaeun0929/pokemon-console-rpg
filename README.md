# 🐾 Python Console Pokémon RPG

**A console-based Pokémon RPG game built with Python, featuring turn-based battles, EXP leveling, and map exploration.**

Python으로 제작한 콘솔 기반 포켓몬 RPG 게임입니다.  
맵을 탐험하며 야생 포켓몬과 전투하고, 경험치를 획득해 포켓몬을 성장시키는 구조로 구현했습니다.

---

## 🎮 주요 기능

### 🗺️ 필드 & 탐험 시스템
- 키보드 입력(W / A / S / D)을 통한 플레이어 이동
- ASCII 문자 기반 콘솔 맵 출력
- 풀숲 진입 시 확률적으로 야생 포켓몬 전투 발생
- 포켓몬 센터에서 체력 회복 가능
- 포탈을 통해 다음 지역으로 이동
- 보스 포켓몬 지역 존재

### ⚔️ 전투 시스템
- 턴제 전투 방식
- 포켓몬별 스킬 사용 가능
- 전투 중 아이템 사용 (회복 / 도망)
- 레벨 차이에 따른 데미지 보정
- 타입 상성(불 / 물 / 풀) 반영

### 📈 성장 시스템
- 공격 1회당 경험치(EXP) 획득
- 경험치 100 도달 시 자동 레벨업
- 전투 승리 시 추가 경험치 지급
- 특정 레벨 도달 시 포켓몬 진화

### 🎒 인벤토리 시스템
- 아이템 획득 및 수량 관리
- 전투 중 인벤토리 사용 가능
- 아이템 타입별 효과 분기 처리

## ▶️ 실행 방법

### 실행 환경
- Python 3.x
- Windows 운영체제  
  (키 입력 처리를 위해 `msvcrt` 모듈 사용)

### 실행 방법
프로젝트 폴더에서 아래 명령어를 실행합니다.

```
bash
python main.py
```

## ⌨️ 조작 방법

### 필드 이동
- `W` : 위로 이동
- `A` : 왼쪽 이동
- `S` : 아래로 이동
- `D` : 오른쪽 이동
- `Q` : 게임 종료

### 전투 중 조작
- `1` : 스킬 1 사용
- `2` : 스킬 2 사용
- `3` : 아이템 사용
- 숫자 입력 후 Enter로 선택
